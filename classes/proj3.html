Fun with Frequencies and Gradients

In this project, we seek to take advantage of the frequency and gradient representations of images to perform visual transformations on them in the spatial domain. This will allow us to perform some really cool tricks with images (from sharpening them to blending them into each other) to create fascinating results.

Hybrid Images

In the hybrid images, we first align the images so that we have an aligned pair of images so that we can have the structures within the image aligned on top of each other for best results. Then, we run one image over a low pass filter to remove some of the details and run the other over a high pass filter and get the detail. Then, we add these together, replacing the detail in the first image with that of the second. This creates a perceptual trick where we perceive the images as both being present. The way that our eyes see means that zoomed out we see the low pass but zoomed in, we see the details. Click on each image to see the image in higher resolution!

Gaussian and Laplacian Stacks

A Gaussian stack simply involves convolving the original image with a Gaussian with increasing sigma at each layer of the stack. It is a stack not a pyramid because we do not downsample at every step. The Laplacian stack subtracts level n+1 of the Gaussian from level n of the Gaussian stack. In this way, the Laplacian stack represents the details in the image at various frequency levels. Note that I made the Laplacian images look more contrasted for more clarity. Multiresolution Blending

Here, we utilize the power of the Gaussian and Laplacian stacks. First, we create a Gaussian stack with the mask image. We do this to account for the fact that the original paper using Gaussian pyramids and that creates additional blending of seams, so we need to simulate that. Then, we create a combined Laplacian stack for the merged image by multiplying the Gaussian layer of the stack of each image by the Gaussian of the mask at every level (or 1 - Gaussian of the mask) depending on the level. Then, we add back the last 2 layers of the Gaussian stack for each image. My favorite result is the Jupiter Sun, which I also applied a tad of darkening to to create a nice effect. Toy Problem

In the case of the toy problem, we seek to solve a system of (2*n) + 1 equations where n is the number of pixels in the image. To solve this, we set up a matrix that is (2*n) + 1 by n. We also set up a matrix with the local gradients for each equation. This gives us a linear system where we are solving for the x in Ax - b = 0. We use the least squares regression technique to find an x vector that we can then reshape back to look like the original image. Note that in the above case, we see an almost perfect reconstruction based on the gradients.. Click on each image to see the image in higher resolution!

Poisson Blending

Poisson blending works by solving a least squares equation for pixels in the area of the mask. It seeks to equalize the gradients of the source image with the target image. Then, by finding a least squares solution, we can find the approximate pixel intensities that best solve these gradient constraints. These may not be the original intensities, but we play on the science of vision and how people notice the gradients of images. The images are (in order) the target image, the source image, the version of the image with the pixels pasted on top, and the final blended image. Note that we include the case of failure in this too. Observe how the blending screws up the arm of the penguin in the blending with the Taj Mahal. Comparing Poisson Blending and Laplacian Blending

We blended the following image with both Poisson Blending and Laplacian Blending - in general, for these more complex cases, the Laplacian blending leaves the ghosting of the previous image in a pretty significant way. This is most noticeable with the turtle, where we see that the result in Poisson blending is pretty nice, even with the gradients of the pool being somewhat complex. However, the Laplacian blending seems to darken the image and leaves a clear trace of ghosting. The same can be seen in the Obama-Penguin blend, where the remnants of the dramatically different pixel intensities are left in the blended image. In less complex scenarios with a half and half mask (like Oraple), Laplacian blending may give better results because it more cleanly blends the seams and maintains similar pixel intensities.
